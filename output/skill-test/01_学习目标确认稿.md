# 7.2《重力 力的示意图》学习目标确认稿

1. 物理观念：学生通过观察水流向低处、跳伞的人落向地面、套圈下落等教材情境，理解重力是物体由于地球吸引而受到的力，知道重力用字母 G 表示，能说明重力方向竖直向下，并能用力的示意图表示物体所受重力。
2. 科学思维：学生通过比较不同质量物体所受重力的数据，能计算 G/m 的比值，归纳物体所受重力与质量成正比，理解公式 G=mg 及 g≈9.8N/kg 的物理意义，并能进行简单计算。
3. 科学探究：学生通过活动 7.1“探究重力与质量的关系”和活动 7.2“判断重力的方向”，能正确使用弹簧测力计测量重力，记录质量和重力数据，分析数据并得出结论；能利用悬线实验判断重力方向。
4. 科学态度与责任：学生通过小组实验、数据交流和生活实例分析，能尊重实验数据，规范使用测量工具，基于证据修正认识，并能认识重力知识在建筑检验、工程测量和生活实践中的应用价值。
