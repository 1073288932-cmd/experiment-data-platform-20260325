# 深度学习问题链设计参考

生成“问题链确认稿”时必须读取本文件。

## 一、设计原则

- 整体性原则：先确定整节课的核心问题，再在每个核心问题下设计子问题链。问题链必须服务于完整的物理知识结构，不能是零散提问。
- 目标性原则：每个问题必须指向学习目标、教学重点、教学难点或核心素养维度。
- 情境性原则：问题必须依托真实现象、教材活动、实验、图片、数据、生活情境或认知冲突，使学生有进入问题的抓手。
- 主体性原则：从学生的学习路径出发设计问题，让学生经历观察、比较、猜想、设计、操作、分析、解释、评价、迁移等学习行为。
- 层次性原则：问题要由浅入深、由现象到本质、由定性到定量、由概念建构到迁移应用。后一个问题应由前一个问题或学生回答自然引出。

## 二、五类问题链

### 1. 核心问题链

作用：统领整节课，指向教学重点和难点。

要求：
- 一节课设计 2 至 4 个核心问题。
- 核心问题不是普通事实性问题，而是整节课的主线。
- 学生在学习前未必能回答，完成相应子问题链后应能回答。
- 每个核心问题下必须有子问题链支撑。

示例：
- 重力是什么？
- 重力的大小与质量有什么关系？
- 如何用力的示意图表示一个力？

### 2. 递进型问题链

作用：帮助学生通过层层推进获得概念或规律。

典型路径：
1. 呈现情境。
2. 观察现象。
3. 分析原因或发现共同特征。
4. 总结归纳概念或规律。

问题特征：
- 后一个问题必须建立在前一个问题的回答基础上。
- 多使用“为什么”“有什么共同点”“能否归纳”“有什么证据”等问题。
- 适用于概念形成、规律归纳、易错点辨析。

### 3. 探究型问题链

作用：按照科学探究流程组织实验活动。

典型路径：
1. 提出问题。
2. 作出假设。
3. 制定计划。
4. 选择器材。
5. 收集证据。
6. 解释问题。
7. 表达交流。

问题特征：
- 教材或教案要求实验时，必须设计探究型问题链。
- 必须包含“选择哪些器材”“实验过程如何设计”“根据什么证据得出结论”。
- 涉及变量控制、数据记录、证据分析、异常数据处理时必须明确提出。

### 4. 过渡型问题链

作用：连接两个知识点、两个活动或两个核心问题。

要求：
- 用在课堂结构可能生硬跳转的位置。
- 问题不一定要求学生马上完整回答，但必须让学生看见“为什么需要进入下一环节”。
- 要承担承上启下作用。

示例：
- 既然重力可以测量，怎样同时表示它的大小、方向和作用点？
- 我们已经得到定性关系，如果要更准确地描述重力与质量的关系，还需要测量哪些量？

### 5. 迁移型问题链

作用：引导学生把概念或规律用于新的真实情境。

要求：
- 放在概念或规律建构之后。
- 情境要真实、新颖或半开放。
- 优先设计解释、设计、决策、评价、改进类问题。
- 可以没有唯一标准答案，但必须能用本课物理知识进行论证。

示例：
- 如何利用本课知识设计一个简易测量或判断工具？
- 如果要让某个装置更安全或更有效，应改变哪些部分？依据是什么？
- 本课规律还能解释哪个课堂外的生活现象？

## 三、确认稿表格格式

向用户展示问题链确认稿时，必须采用下列表格结构：

| 教学情境组织流程 | 问题链设计（如果有预设回答可以注明） | 问题链类型 |
|---|---|---|
| 核心问题1：…… || 核心问题链 |
| 情境1：…… | 1-1-1 …… | 递进型问题链 |
| 观察/活动/探究：…… | 1-1-2 …… | 递进型问题链 / 探究型问题链 |
| 过渡：…… | 1-2-1 …… | 过渡型问题链 |
| 核心问题2：…… || 核心问题链 |
| 探究1：…… | 2-1-1 …… | 探究型问题链 |
| 迁移情境：…… | 2-2-1 …… | 迁移型问题链 |

## 四、质量底线

出现以下情况，问题链不合格：

- 只是列出 10 个互不关联的问题。
- 大多数问题只需回忆即可回答，不需要证据、推理或解释。
- 没有核心问题。
- 核心问题下没有子问题链。
- 没有说明问题来源于教材、上传教案、生活情境或实验数据。
- 实验课缺少探究型问题链。
- 缺少迁移型问题链。
- 问题之间没有承接关系。
- 忽略上传教案中已有的优质活动。

优先使用这些行为动词设计问题：观察、比较、预测、解释、设计、测量、记录、分析、归纳、评价、迁移、改进。
